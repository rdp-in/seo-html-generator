Generates html code for Yoast SEO options. Usefull when migrating Wordpress SEO to Static HTMl sites.

== Description ==

The plugin was primarily developed to suffice the requirement of migrating wordpress Yoast SEO to a static HTMl site.
It basically generates the SEO content for every page, just like the YOAST SEO plugin.
This content can then be pasted into the static html web page head tags.

The plugin provides option on the WP pages page.
Navigate to WP admin -> Pages.
You'll find an option that says "Generate SEO html" which will do the job.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/seo-html-generator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to WP Admin -> Pages page.
